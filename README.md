# اتاق فرار هاتف — Hatef's Escape Room

یک بازی اتاق‌فرار آموزشیِ تک‌نفره، مرورگری و کاملاً فارسی (RTL) که با فرار از «ذهن» یک ابرهوش‌مصنوعی، مسیر کامل کار یک مدل زبانی (LLM) را تعاملی آموزش می‌دهد. ساخته‌شده برای کلاس زنده (تا ۶۰ دانشجو، هرکس روی دستگاه خودش).

پنج لایه: **توکن‌سازی → Embedding → Attention → تولید توکن → مهندسی پرامپت**. شصت دقیقه، پنج کلید.

## اجرا

```bash
npm install
npm run dev
# http://localhost:3000
```

بازی از همان اجرای اول کامل و قابل‌بازی است — بدون نیاز به تصاویر یا سرویس خارجی (fallback گرادیان + موسیقی تولیدشونده‌ی Tone.js).

## پیکربندی

`.env.example` را به `.env.local` کپی کن:

- `ANTHROPIC_API_KEY` — برای لایه ۵ (مهندسی پرامپت) که پرامپت بازیکن را به Claude API می‌فرستد. **اگر تنظیم نشود**، لایه ۵ در «حالت آفلاین» با یک ارزیاب لوکال کار می‌کند تا بازی همیشه قابل‌اتمام باشد.

## تصاویر (اختیاری)

فایل‌های زیر را در `public/images/` بگذار (هرکدام نبود، به گرادیان fallback می‌شود):
`room.png`, `hatef-orb.png`, `layer-1.png`…`layer-5.png`, `win.png`, `lose.png`.

## صدا (اختیاری)

اگر `public/audio/ambient.mp3` وجود داشته باشد، به‌جای موسیقی تولیدشونده پخش می‌شود.

## استک

Next.js (App Router) · TypeScript · Tailwind CSS · Framer Motion · Tone.js · Zustand · `@anthropic-ai/sdk`. آماده‌ی دیپلوی روی Vercel.

## دیپلوی روی Vercel

پروژه را به Vercel وصل کن و `ANTHROPIC_API_KEY` را در Environment Variables تنظیم کن (اختیاری). بدون آن هم کامل کار می‌کند.
